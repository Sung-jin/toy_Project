# [Liquibase](https://www.liquibase.org/)

* database 의 스키마 변경을 관리해주는 오픈소스.
* XML|YAML|JSON|SQL 모든 포멧의 파일로 관리 할 수 있다.
* command line / maven / gradle 등 다양한 방식으로 사용 할 수 있다.
